### Getting Started

- Clone this repo
- Run `bundle install`
- Run `npm install`
- Create a `.env` file with your own personal Marvel API keys

### Running tests
#### Rspec
- `bundle exec rspec`

#### Mocha
- `npm test`

### Running locally
- To run locally with caching create an empty file `tmp/caching-dev/txt`
- `bundle exec rails server`
- visit http://localhost:3000
